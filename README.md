Install/Run Locally
-------------------
open Terminal

Install Node Modules

```npm install```

This will start up [webpack-dev-server](https://webpack.github.io/docs/webpack-dev-server.html) on [localhost:8080](http://localhost:8080)

```npm run dev```


-- OR --

Build and Run
```
npm i serve -g
npm run build
serve dist/
```
